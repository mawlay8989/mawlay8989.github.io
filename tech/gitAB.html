What is Git?

Git is a Version Control System that allows developers to collaborate on coding projects and track changes over time. It was created by Linus Torvalds (the creator of Linux) in 2005 and is used by the majority of developers worldwide.

Installation & Configuration

Now let us check if Git was installed correctly by executing the following command :

$ git --version
git version 2.37.2

Then, we set a name and email for our future Git repositories : 

git config --global user.name "John Doe"
git config --global user.email "john.doe@gmail.com"
We use the --global option to make sure our name and email are used for every Git repository on our computer.
